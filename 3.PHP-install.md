## Install PHP-5.6.40 on Rocky Linux 8:

_Enable PowerTools (needed by many PHP packages):_
```
dnf install -y epel-release
dnf config-manager --set-enabled powertools
```


_Install Remi repository for Enterprise Linux 8:_
```
dnf install -y https://rpms.remirepo.net/enterprise/remi-release-8.rpm
```


_Refresh repositories:_
```
dnf makecache
```


### Install PHP 5.6 Packages:

Remi’s packages for PHP-5.6 on EL8 use a software collections style prefix (`php56-php-*`) so that multiple PHP versions can coexist: 

```
dnf install php56 -y

or,

yum install php56 php56-php-common php56-php-mysqlnd php56-php-pgsql php56-php-pecl-mongo php56-php-pdo php56-php-pecl-memcache php56-php-pecl-memcached php56-php-gd php56-php-xml php56-php-mbstring php56-php-mcrypt php56-php-pecl-apcu php56-php-cli php56-php-pear php56-php-curl php56-php-mysql php56-php-ldap php56-php-zip php56-php-fileinfo -y
```


```
dnf install php56-php-fpm -y
```


```
systemctl enable php56-php-fpm
systemctl restart php56-php-fpm
```

_Once installed, the PHP 5.6 binaries will be under:_
```
/opt/remi/php56/root/usr/bin/php -v
```

